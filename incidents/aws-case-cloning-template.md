# AWS Support Case Cloning Template

## Scope

Create a reusable AWS Support case template for cloning issue #8 into a new customer-service support request.

## Case Type

Customer service

## Console Path

https://support.console.aws.amazon.com/support/home?issueType=customer-service#/

## Request Statement

We need assistance scoping and creating a repeatable AWS Support case template for issue #8.

The template should help us open a new customer-service case with clear context, affected AWS services, account details, requested outcome, and any supporting steps already taken.

## Template Fields

- Case title
- AWS account ID
- Environment or workload
- Affected services
- Business impact
- Problem statement
- Timeline
- Troubleshooting already performed
- Requested AWS Support action
- Attachments or supporting references
